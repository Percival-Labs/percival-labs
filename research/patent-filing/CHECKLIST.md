## Filing Day Sequence

### Before Filing
- [ ] Confirm Percival Labs LLC is active with WA Secretary of State
- [ ] Fill in address/phone/email in cover sheet
- [ ] Verify micro entity eligibility (income < ~$242K, no prior patents)
- [ ] Sign IP assignment agreement (inventor → LLC)
- [ ] Convert specification to PDF (`pandoc` or print-to-PDF)
- [ ] Convert cover sheet to PDF
- [ ] Convert micro entity cert to PDF
- [ ] Create USPTO.gov account at https://account.uspto.gov

### Filing
- [ ] Log in to https://patentcenter.uspto.gov
- [ ] Start new provisional application submission
- [ ] Upload specification PDF
- [ ] Fill in electronic cover sheet (or upload PDF)
- [ ] Select "micro entity" and submit certification
- [ ] Pay $320 fee
- [ ] Download and save filing receipt

### After Filing
- [ ] Record application number: _______________
- [ ] Record filing date (priority date): _______________
- [ ] Record confirmation number: _______________
- [ ] Record assignment at USPTO EPAS ($50): https://epas.uspto.gov
- [ ] Add "Patent Pending" notice to relevant products
- [ ] Set calendar reminder: 12-month non-provisional deadline
- [ ] Store all documents in secure location
